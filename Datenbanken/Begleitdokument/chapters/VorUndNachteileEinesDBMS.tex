\section{Vor- und Nachteile Eines DBMS}

Für das Wahlinformationssystem muss entschieden werden, ob ein Datenbank Management System (DBMS) verwendet werden soll. Dafür müssen die Vor- und Nachteile eines solchen abgewägt werden. \bigskip

\textbf{Vorteile:}
\begin{itemize}
    \item Verhindern von Redundanz und Inkonsistenz
    \item Einheitliche Modellierung der Informationen und damit erweiterte Zu\-griffs\-mög\-lich\-kei\-ten
    \item Möglichkeit des Mehrbenutzerbetriebes
    \item Vorbeugung gegen Datenverlust
    \item Durchsetzung von Integritätsbedingungen
    \item Durchsetzung von Sicherheitsmechanismen
    \item Niedrigere Entwicklungskosten für neue Anwendungsprogramme
\end{itemize}



\textbf{Nachteile:}
\begin{itemize}
    \item Kosten für Anschaffung und Betrieb des DBMS
    \item Gegenüber Papierlösungen: Angreifbarkeit durch Computerviren
    \item Mögliche Abhängigkeit von einer einzelnen Software
\end{itemize}

Insgesamt überwiegen die Vorteile eines DBMS.
