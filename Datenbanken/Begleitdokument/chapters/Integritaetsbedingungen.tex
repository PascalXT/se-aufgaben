\section{Integritätsbedingungen}

Die Datenbank sollte folgende Integritätsbedingunen erfüllen:

\begin{itemize}
    \item Jeder Listenplatz darf maximal einmal vergeben werden.
    \item Die Funktionalitäten müssen eingehalten werden
    \item Eine Erststimme kann nur einem Kandidaten gegeben werden, der in dem zu\-ge\-höri\-gen Wahlkreis für ein Direktmandat kandidiert.
    \item Eine Zweitstimme kann nur einer Partei gegeben werden, die eine Landesliste in dem zu\-ge\-höri\-gen Bundesland hat.
    \item Jede Partei darf in jedem Bundesland maximal eine Landesliste haben.
    \item Ein Kandidat, der Mitglied einer Partei ist, darf nicht für eine andere Partei kandidieren.
\end{itemize}
